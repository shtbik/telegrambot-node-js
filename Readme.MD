# telegrambotNode
Небольшой бот-напоминалка для телеграма на Node JS.<br/>
username: @AlexanderShtykovBot<br/>

Пример работы:<br/>
Вы: /напомни мне уйти с работы в 18:15<br/>

(18:15)<br/>
Бот: Напоминаю, что Вы должный уйти с работы сейчас.<br/>
